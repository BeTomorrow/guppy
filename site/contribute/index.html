Develop

If you want to help with development, this page contains some resources to help set up your environment for running, testing, and understanding the code. I also try to be responsive to issues raised via github. Feel free to raise questions about development there, as issues with this documentation.

  • To get started, run: 
    $ git clone https://github.com/daniel3735928559/guppy
$ cd guppy
$ npm install --only=dev
$ ./make -d
  • For future development plans and direction, check the roadmap.
  • For current outstanding issues, check the issue tracker on github.
  • For an introduction to the editor's internal algorithms, see here, and also here for a description of the XML format used internally by the editor.
