Mathematics input widget for the web






LaTeX Text ASCIIMath JSON XML

Human-friendly

  • WYSIWYG, backslashes optional.
  • Keyboard only; optional mouse interface.
  • Mobile keyboard.
  • LaTeX or plain text output.
  • Supports matrices and custom symbols.

Computer-friendly

  • Expressions stored in semantic XML.
  • XML or JSON output.
  • Generate and evaluate syntax trees.

Developer-friendly

  • Configure options globally or per editor.
  • Flexible API. Scriptable.
  • Style using CSS
